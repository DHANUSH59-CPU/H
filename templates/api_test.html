Available API Endpoints
GET /health

Basic health check of the API service.

Response: 
{
  "status": "healthy",
  "timestamp": "2025-09-18T14:08:21.972372",
  "service": "ML Prediction API",
  "version": "1.0.0"
}
GET /health/detailed

Detailed health check with component status information.

Response: 
{
  "status": "healthy",
  "timestamp": "2025-09-18T14:08:21.972372",
  "service": "ML Prediction API",
  "version": "1.0.0",
  "components": {
    "models": {"status": "healthy", "total_models": 4},
    "monitoring": {"status": "healthy"},
    "retraining": {"status": "healthy"},
    "filesystem": {"status": "healthy"}
  }
}
GET /health/readiness

Readiness check - verifies service is ready to serve requests.

GET /health/liveness

Liveness check - verifies service is alive and responding.

POST /predict

Make predictions using the active model.

Request Body: 
{
  "features": {
    "feature_1": 1.5,
    "feature_2": 2.0,
    "feature_3": 1.2,
    "feature_4": 0.8
  },
  "model_id": "optional_specific_model_id"
}
Response: 
{
  "prediction": "class_a",
  "model_id": "random_forest_clf_20250918_135521",
  "model_type": "random_forest_classifier",
  "confidence": 0.85,
  "response_time_ms": 150,
  "request_id": "req_1726665501972"
}
GET /metrics

Get comprehensive monitoring metrics and dashboard data.

Query Parameters:
  • model_id (optional) - Filter metrics for specific model
  • hours (optional) - Time range in hours (default: 24)
POST /model/retrain

Trigger manual model retraining.

Request Body: 
{
  "model_id": "model_identifier",
  "notes": "optional notes about retraining"
}
GET /alerts

Get active alerts for monitoring.

Query Parameters:
  • model_id (optional) - Filter alerts for specific model
Code Examples
Python (requests)
import requests
import json

# API base URL
base_url = "http://localhost:5000"

# Health check
response = requests.get(f"{base_url}/health")
print("Health:", response.json())

# Get model status
response = requests.get(f"{base_url}/model/status")
print("Models:", response.json())

# Make prediction
data = {
    "features": {
        "feature_1": 1.5,
        "feature_2": 2.0,
        "feature_3": 1.2,
        "feature_4": 0.8
    }
}

response = requests.post(
    f"{base_url}/predict",
    json=data,
    headers={'Content-Type': 'application/json'}
)

if response.status_code == 200:
    result = response.json()
    print(f"Prediction: {result['prediction']}")
    print(f"Confidence: {result.get('confidence', 'N/A')}")
else:
    print(f"Error: {response.text}")
JavaScript (fetch)
// API base URL
const baseUrl = "http://localhost:5000";

// Health check
async function checkHealth() {
    const response = await fetch(`${baseUrl}/health`);
    const data = await response.json();
    console.log("Health:", data);
}

// Make prediction
async function makePrediction() {
    const data = {
        features: {
            feature_1: 1.5,
            feature_2: 2.0,
            feature_3: 1.2,
            feature_4: 0.8
        }
    };

    try {
        const response = await fetch(`${baseUrl}/predict`, {
            method: 'POST',
            headers: {
                'Content-Type': 'application/json'
            },
            body: JSON.stringify(data)
        });

        if (response.ok) {
            const result = await response.json();
            console.log(`Prediction: ${result.prediction}`);
            console.log(`Confidence: ${result.confidence || 'N/A'}`);
        } else {
            const error = await response.json();
            console.error('Error:', error);
        }
    } catch (error) {
        console.error('Network error:', error);
    }
}
